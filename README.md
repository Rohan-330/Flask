# Flask Learning Journey 🚀
This repository contains a collection of mini-projects developed during my **FY B.Sc. IT** studies. Each project focuses on implementing logic-based problems through a web interface using the **Flask** framework.
## 📁 Project Structure
The repository is organized by specific applications, each containing the necessary Flask routing and logic.
 * **calculator-app**: Basic arithmetic operations.
 * **matrix-addition/subtraction/multiplication**: Handling 2D arrays in Python.
 * **even-number/odd-number/leap-year**: Conditional logic apps.
 * **string-concat**: String manipulation and display.
 * **swap-app**: Variable value swapping logic.
 * **greeting-app**: User input and dynamic response.
## 🎯 Learning Objectives
 * Understanding the **Flask** micro-framework architecture.
 * Mapping Python logic to web routes using @app.route().
 * Handling **GET** and **POST** requests from web forms.
 * Rendering dynamic data using **Jinja2** templates.
 * Structuring Python projects for better readability.
## 🛠️ How to Run the Projects
Follow these steps to run any application on your local machine:
 1. **Clone the Repository**
   ```bash
   git clone https://github.com/Rohan-330/Flask.git
   cd Flask
   
   ```
 2. **Navigate to a Project Folder**
   ```bash
   cd swap-app  # Or any other folder name
   
   ```
 3. **Install Flask** (If not already installed)
   ```bash
   pip install flask
   
   ```
 4. **Run the Application**
   ```bash
   python app.py
   
   ```
 5. **Access in Browser**
   Open your browser and go to: http://127.0.0.1:5000/
